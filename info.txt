Fixed Point Iteration Calculator
================================

A clean-architecture Java Swing application that performs Simple Fixed Point Iteration on equations of the form:

    f(x) = 0

The system transforms the equation into a suitable g(x), performs iterations, and displays results in a structured table.

Class Interaction Summary
=========================

1. User Interface Layer (ui/)
----------------------------

InputPanel
- Accepts user input: f(x), initial guess x0, tolerance percentage.
- Provides Compute and Clear Output buttons.

CalculatorFrame
- Coordinates UI behavior.
- Uses EquationConverter to convert f(x) to g(x).
- Uses FixedPointSolver to compute iterations.
- Displays derived g(x), iteration table, and convergence info.

2. Conversion Layer (core/converter/)
-------------------------------------

EquationConverter
- Converts user input f(x) into g(x).
- Uses polynomial rearrangement.
- Uses pattern matching for expressions like e^(-x) - x.
- Falls back to relaxation form g(x) = x - λ·f(x).

PolynomialParser
- Parses polynomial terms into coefficient maps.
- Supports automatic symbolic rearrangement for g(x).

3. Evaluation Layer (core/evaluator/)
-------------------------------------

Evaluator (interface)
- Defines evaluate(x), isValid(), and getExpression().

Exp4jEvaluator
- Uses exp4j to evaluate expressions.
- Supports powers (^), exp(), trigonometric functions, logs, etc.

4. Solver Layer (core/solver/)
------------------------------

FixedPointSolver
- Performs simple fixed-point iteration.
- Computes approximate and relative percent errors.
- Produces iteration rows and convergence result.

5. Model Layer (core/model/)
----------------------------

IterationResult
- Contains:
  - derived g(x) expression
  - list of iteration rows
  - convergence flag
  - optional error message

Iteration Process Description
=============================

1. Convert f(x) into g(x) using EquationConverter.
2. Apply iteration formula x_{i+1} = g(x_i).
3. Compute approximate and relative percent errors.
4. Stop when ea <= tolerance or max iterations reached.
5. Return results via IterationResult.

Build and Run Instructions
==========================

Build the project:

    mvn clean install

Run:

    mvn exec:java

Key Features
============

- Clean code and SOLID structure.
- Exp4j-based evaluator for modern Java versions.
- Robust equation conversion.
- Detailed iteration output.
- Clear separation of concerns across layers.

Extensibility
=============

- Add custom evaluators by implementing Evaluator.
- Add new solvers under core/solver/.
- Extend UI under ui/.
